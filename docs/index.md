# VEILUX

**A featherweight, privacy-first, AI-native Layer-1 blockchain written in Rust.**

Launch a network, deploy Solidity, move private value between named parties, run
verifiable on-chain AI, and slash cheaters — from a single self-contained binary.

[View on GitHub](https://github.com/VeiluxLabs/Veilux-Binary){: .btn }
[Download a release](https://github.com/VeiluxLabs/Veilux-Binary/releases){: .btn }

> **Honest status:** VEILUX is a fully functional, test-covered **testnet-grade
> core**. It has **not** had an independent third-party audit and has not run a
> public mainnet. Do not secure real value with it yet. See the
> [internal audit](audit-2026-06.md) and [security model](security.md).
