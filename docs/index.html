What it does

One binary, a full Layer-1

No JVM, no sprawling toolchain. Everything is a self-contained Rust workspace with a pluggable “Prism” model for capabilities.

🔐

Banking-grade privacy

Each event is sealed per-party with ChaCha20-Poly1305. All nodes agree on a Merkle root of blinded data — only stakeholders decrypt their own events.

⟠

Real EVM

A from-scratch EVM: deploy & call Solidity over eth_*, inter-contract calls, CREATE/CREATE2, and precompiles. MetaMask & ethers.js just work.

🧠

On-chain AI

Register models and run deterministic inference (or a local LLM via Ollama). Results are committed and auditable, with large outputs offloaded to content-addressed storage.

⚖️

BFT + slashing

Aurora BFT finality with proposer failover. Equivocation and private-root divergence are caught cryptographically and slashed on-chain.

🧩

Nine Prisms

Token, NFT, Contract (PhotonVM), AI, Bridge, Staking, Oracle, Storage, Confidential — install what you need, build your own.

🛠️

Rust & TS SDKs

JSON-RPC, WebSocket subscriptions, a web explorer, and first-class Rust + TypeScript SDKs for building applications.

Quickstart

Running in 60 seconds

Build the binary and watch a full chain do its thing — AI, privacy, tokens, NFTs, and a contract — end to end.

# build the single binary
$ cargo build --release

# see the kernel + installed prisms
$ ./target/release/veilux info

# run the end-to-end demo (AI · privacy · token · NFT · contract)
$ ./target/release/veilux demo
  contract deployed at 0xcd69...7058
  call result: 111 + 222 = Some(333)

# launch a dev RPC node + Ethereum-compatible endpoint
$ ./target/release/veilux serve 127.0.0.1:8645 --eth-rpc 127.0.0.1:8655
